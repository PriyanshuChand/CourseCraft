## Description
CourseCraft is a full-stack course selling web application that allows users to browse courses, enroll in them, and manage their learning through a simple dashboard. It also includes admin functionality for creating and managing courses.

This project is built to demonstrate real-world web development concepts such as authentication, CRUD operations, REST APIs, and frontendâ€“backend integration.

## Features
- User authentication (signup/login)
- Course listing and course details
- Course enrollment flow
- User dashboard
- Admin panel for course management
- Secure APIs and database integration

## Tech Stack
- Frontend: HTML, CSS, JavaScript, React
- Backend: Node.js, Express
- Database: MongoDB 
- Authentication: JWT / Sessions

## Status
Core features are under development. More functionality and UI improvements will be added incrementally.

## Learning Goals
- Build a real-world full-stack application
- Practice clean backend APIs
- Improve database design and authentication handling
- Gain experience structuring a production-style project
